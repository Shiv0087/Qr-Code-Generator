# Qr-Code-Generator

# 🔳 QR Code Generator – Text to QR with Download & Share

Welcome to the **QR Code Generator** – a premium, minimal, and powerful web tool that allows you to convert any text into a high-quality QR Code in seconds!

🌐 [Live Demo](https://matrixCoderX.github.io/Qr-Code-Generator)

---

## ✨ Features

- ✅ Convert **any text** instantly into a **QR Code**
- 🎨 Beautiful, modern, and premium UI
- ⬇️ **Download QR Code** as image (PNG)
- 📤 **Share QR Code** directly (via WhatsApp, Telegram, etc.)
- 🌓 Smooth animations & responsive design
- 💻 Works perfectly on **mobile & desktop**

## 🚀 How to Use

1. Enter the text you want to convert into a QR code
2. Click on `Generate QR`
3. Click `Download` to save the QR
4. Click `Share` to open sharing options
